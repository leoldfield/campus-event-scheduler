# Campus Event Scheduler

UALR 2026 Capstone Project (Team Bravo)

This project is a website for a campus event where students can easily find and register for the event, and organizations can create and manage their events in one place.

## Why are we building this?

The students are missing out on these events due to the information being spread out all over emails, social media, and different departmental pages.

## What can the system do? (MVP)

- Users can create an account and log in
- Users can browse events
- Users can search/filter events
- Users can register for an event
- Organizations/admins can create and manage events

## Tech Stack (planned)

- Frontend: HTML / CSS / JavaScript 
- Backend: Node.js 
- Database: MySQL
- Version control: GitHub
